# Linux浏览器报错

Chromium问题，一般是你的浏览器没安装，有时候Linux是不带浏览器的，安装一下浏览器就行了

sudo apt update  #更新仓库
sudo apt install chromium  #安装浏览器
chromium -version   #检查浏览器版本
sudo apt purge --remove chromium  #卸载浏览器

yun install chromium #安装浏览器

去浏览器下载吧 
下载直达(https://download-chromium.appspot.com/)

如果安装报错就卸载一遍浏览器重新安装

# git clone / pull 不了或下载的慢怎么办

  • 你需要魔法上网
  • 你已经有梯子了但是你的git不愿意走代理怎么办
  • 那就尝试配置一下全局代理吧
git config --global https.proxy 127.0.0.1:7890 #设置https代理
git config --global http.proxy 127.0.0.1:7890 #设置http代理
git config --global --unset https.proxy #删除代理
git config --global --unset http.proxy #删除代理

127.0.0.1:7890是指向本地的，如果你有其他代理地址或者网络就填那个吧

  • 挂上代理后会出现无法clone Gitee的情况，请注意关闭代理

  • 修改一下代码，自定义端口
bind_port = 17000   #自定义外网端口，你想改什么都可以

# Frpc配置，官方文档open in new window

# frpc.ini
[common]
server_addr = x.x.x.x  #你服务器的IP地址或者域名
server_port = 17000     #连接端口与服务端（frps）一致

[ssh]
type = tcp             #链接协议
local_ip = 127.0.0.1   #链接地址
local_port = 22        #不用修改
remote_port = 6000     #向服务器映射的端口，需要在服务器防火墙放行

搭建完成后，进入路由器管理放行端口：17000（自定义外网端口）和对方设置6000（Frpc）的端口

# 安装ffmpeg

# Linux安装

apt install ffmpeg

git clone https://git.ffmpeg.org/ffmpeg.git ffmpeg #克隆仓库
cd ffmpeg      #进入目录
./configure    #检测你的安装平台的目标特征
make           #编译命令
make install   #编译安装
cp ffmpeg /usr/bin/ffmpeg  #将ffmpeg剪切至/usr/bin/目录下
cp ffmpeg /usr/bin/ffprobe #将ffprobe剪切至/usr/bin/目录下
ffmpeg -version   #检查版本

# nb2安装教程

官方安装教程open in new window

  • 新建一个文件夹，启动powershell

  • 安装pipx

pip install --user pipx
pipx ensurepath
  • 安装脚手架
pipx install nb-cli
  • 启动nb安装程序，真的只需要输入下面这个....
nb

  • 选择新建机器人

  • 协议安装请选择onebot v11

  • # nb2配置gocq文件

    • nb2使用反向 Websocket协议，所以gocq安装时需要选择反向 Websocket 协议
    • 修改gocq配置文件config.yml，找到以下配置，将其修改成这样
post-format: array
universal: ws://127.0.0.1:8080/onebot/v11/ws/
